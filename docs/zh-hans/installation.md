## 安装

TanStack Form 兼容多种前端框架，包括 React、Vue 和 Solid。要结合您选用的框架使用 TanStack Form，请通过您首选的包管理器安装相应的适配器：

### React 示例

```bash
# npm
$ npm i @tanstack/react-form
# pnpm
$ pnpm add @tanstack/react-form
# bun
$ bun add @tanstack/react-form
# yarn
$ yarn add @tanstack/react-form
```

### Vue 示例
